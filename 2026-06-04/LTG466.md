## Why This Session

MCP (Model Context Protocol) is the new standard for connecting agents to tools. This builds an MCP server from scratch using the **C# SDK** — your language. As a .NET community leader, this is content you'll want to teach at user groups.

## Abstract

The Model Context Protocol (MCP) is all the rage and rightly so. As the USB-C for AI, it is an incredibly powerful tool that acts as a bridge between real-world systems and LLMs. But how do you get started with MCP and more importantly: how do you build your own MCP server? In this session we'll build an MCP server from scratch using the MCP C# SDK. We'll look at how to build tools, prompts and resources and how to leverage these with our own AI agent in Microsoft Foundry.
