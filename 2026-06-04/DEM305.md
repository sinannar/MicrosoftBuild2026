## Why This Session

As a GitHub partner who works across CLI, VS Code, and cloud environments — this shows how Copilot maintains context across all of them. Relevant to your daily workflow and client advisory on GitHub adoption.

## Abstract

Modern development no longer happens in one place. You might start in your terminal, step away, and pick things back up from another device or let work continue in the cloud. This session explores using GitHub Copilot across environments, from local CLI sessions to full cloud execution with remote execution environments. See how Copilot runs commands, iterates on changes, and continues work without losing context, while you stay in control.
