## Why This Session

Your #1 lab pick. You use PostgreSQL daily with Entity Framework and this takes you hands-on with Azure HorizonDB — the new cloud-native PostgreSQL that does RAG, hybrid search, and in-database AI without external vector stores. No glue code. This is your EF + PostgreSQL expertise evolving into the AI era.

## Abstract

In this lab, you'll build a production-ready AI agent using Azure HorizonDB as a unified data, retrieval, and intelligence layer. Implement RAG, hybrid semantic and keyword search, and agentic workflows without stitching external services. Use DiskANN advanced filtering and in-database AI model management to run LLMs and embeddings directly in the database — no vector stores or glue code. Ideal for developers and architects who want simpler stacks, higher performance, and faster AI delivery.
