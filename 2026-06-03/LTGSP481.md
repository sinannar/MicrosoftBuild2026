## Why This Session

Combines .NET Web APIs with durable orchestration (Temporal) for autonomous agents. This pattern — long-running tasks, failure management, sandboxed execution — is directly relevant to building production agent systems for your consulting clients.

## Abstract

Modern AI can generate code—but building agents that autonomously fix bugs, run tests, and create pull requests requires real systems engineering. Dive into a hands-on architecture for coding agents built with .NET Web APIs, Temporal workflows, and AI-powered workers. See how to orchestrate long-running tasks, manage failures, and securely execute code in sandboxed environments to deliver production-ready autonomous agents.
